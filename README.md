# Project-1-Aviation-
The objective of this project is to identify and highlight factors that contribute to flight delays across different airlines
, states, and days of the week.

## Overview of project

## Loaded Data:-

Loaded data to power query editor
Checked Data Types , Promoted Headers, Replaced Null Values with 0, Trimmed and cleaned data.

## Data Modeling:-  

Integrate multiple Excel and SQL data sources, and establish relationships for analysis in Power BI or Tableau.

# Key Performance indicators:-

1.Weekday Vs Weekend total flights statistics

2.Total number of cancelled flights on first date of every month

3.Week wise, State wise and City wise statistics of delay of flights with airline details

4.Number of airlines with No departure/arrival delay with distance covered between 2500 and 3000

5.Departure/arrival delay status by departure/arrival time.

6.Arrival Delay and depature delay of flights by ariline.
   
7.Top 10 Cities with no delay State wise.

# Insights

Identified key drivers of flight delays across 5.8M+ records, enabling actionable insights into operational efficiency, scheduling optimization, and network reliability.
1. The weekday flight volume is significantly higher than weekends, which directly correlates with increased arrival and departure delays.
   
3. JetBlue Airways recorded a notably higher number of cancelled flights on the first day of each month compared to other days.
   
5. The KPIs show ~2.09M arrival-delayed flights and ~2.13M departure-delayed flights, indicating a strong dependency between the two.

6. Southwest Airlines consistently recorded the highest arrival and departure delays, while Virgin America and Hawaiian Airlines showed better on-time performance.
